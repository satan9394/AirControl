# AirControl 项目报告

## 功能概览
- 摄像头采集与手部关键点识别（MediaPipe Hands）
- 光标移动（食指指尖映射 + OneEuro 平滑 + 自适应增益）
- 单击（短闭合 pinch）、双击（拇指同时夹食指/中指的专用手势）
- 右键（食指+中指伸出下击上扬）、拖拽（长按 pinch）、滚动（握拳上下）
- 菜单与校准（张开手掌保持、OK 长按）
- 多显示器支持（锁定到目标显示器或虚拟屏全域映射）
- 桌面透明覆盖层：轨迹、指针圆点、操作指南文本
- 手部骨架参考：双手彩色分指骨架、端点高亮、中心缩放以减少遮挡
- 设置面板：ROI、死区、平滑、增益、镜像 X/Y、目标显示器、轨迹参数等

## 交互设计
- 手势全集基于“形状 + 状态机”，避免“两次单击”时间窗不稳
- 事件优先级：拖拽/滚动/菜单 等模式避免与单击/双击冲突
- 控制手选择：优先使用右手（有 handedness 时），否则使用第一只手；骨架显示支持双手

## 技术要点
- OneEuro 平滑与速度自适应增益，提高慢速稳定与快速灵敏
- ROI + 边缘死区：减少边缘误触，支持校准更新中心
- Win32 `SetCursorPos` 驱动鼠标，降低延迟
- DPI 感知处理，减少 Qt 提示并保证坐标正确

## 目录结构
- `aircontrol/` 摄像头、手部追踪、手势引擎、映射、OS 适配、UI、监视器、覆盖层
- `main.py` 入口整合与实时循环
- `requirements.txt` 依赖
- `操作指南.md` 操作说明
- `项目报告.md` 本报告

## 使用指南
- 安装依赖：`pip install -r requirements.txt`
- 启动：`python main.py`
- 设置面板：按 `s` 打开/隐藏，应用后即时生效并写入 `config.json`
- 建议调参：先 `OK` 校准 → 调 ROI → 死区 → 平滑/增益 → 镜像 X/Y
